# Food-Delivery-Website
A fully functional Food Delivery Website with multiple sections and interactive features.   

## 📌 Features:  
✅ Sections: 
- 🏠 Home  
- 📖 About  
- 📞 Contact  
- 🍽️ Menu
- 🛍️ Product
- 📝 Blog  
- 🔐 Login / Forgot Password / Create Account  

## ✅ Functionalities:  
- ⭐ Add Reviews  
- 🛒 Add/Remove Items from Cart  
- 🔍 View Menu Items  
- 📩 Contact Form Submission  
- 🖊️ Blog Section  

## 💻 Tech Stack:
    HTML5
    CSS
    JavaScript
